```

        Mass Bin Checker
================================

Usage: t.pl [comando]
[+] Comandos:
--bin  [Checa apenas 1 bin]
--help [Ajuda com os comandos]
--list [Seleciona sua lista de bin a ser checada]
--save [Onde as bins serao salvas]
```
